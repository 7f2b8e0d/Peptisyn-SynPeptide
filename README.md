### Deploy to GitHub Pages
1. Push this repository to GitHub.
2. Open Settings → Pages.
3. Under "Build and deployment", set Source = "Deploy from a branch", Branch = `main`, Folder = `/ (root)`.
4. Save. Your site will be published at `https://<username>.github.io/<repo>/`.

A `.nojekyll` file is included so all folders and assets are served as-is.
